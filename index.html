Welcome to the Health Check API

This is the landing page for the Health Check API hosted on Fly.io.

Use the endpoint /process_survey to send survey data as a JSON payload.

Example Payload:

{
  "score": [4, 3, 2, 1]
}

Expected Response:

{
  "mean_score": 2.5,
  "min_score": 1,
  "max_score": 4,
  "count": 4
}
